英雄名称：Anti-Mage 敌法师

【英雄简介】
攻击敌人时会燃烧敌人的魔法
只要给了敌法师积累力量的机会，他的碾压将无法阻挡。进，他会枯竭敌人的魔力，退，他可短距离传送。想将他逼入困境绝非易事。

【命石】
1.灭法之镜
法术反制成功反制技能后会使敌人立刻被燃烧该技能魔法消耗的一定百分比，并且根据燃烧的魔法消耗造成伤害。
法术反制
法术反制会使施法者被燃烧该技能魔法消耗的一定百分比，并且造成100%燃烧魔法的伤害，最高为每次反弹技能造成400点伤害。
燃烧魔耗百分比：
150% / 200% / 250% / 300%
燃烧魔耗百分比：
0% / 0% / 0% / 0%

2.法力渴望
敌法师在敌人的魔法较低时获得攻击力。
法力虚空
敌方英雄的魔法值低于60%时会被动提供攻击力，而且魔法值越低，提升效果越大。

如果敌人的魔法值低于15%，敌法师获得攻击力加成的最大值，以及该英雄的视野和真实视域。会施加给2500范围内的敌方英雄。每个英雄提供的加成效果可以叠加。
法力渴望攻击力加成上限：
20 / 45 / 70
法力渴望攻击力加成上限：
0 / 0 / 0
